# Prompt-like-a-pro materials for teachers

This repository now includes a small OECD-focused teaching pack for higher education faculty.

## Included files

- `updated_oecd_ai_rubric_5_point.docx` - 5-point rubric for implementing OECD AI principles in higher education classrooms
- `updated_30min_lesson_plan.docx` - 30-minute hands-on lesson plan using the 5E model
- `updated_OECD_AI_Principles_Engineering_Education.docx` - assessment question bank for faculty development and classroom use
- `updated_3Step_Learning_Scaffold.docx` - three-step learning scaffold from foundational knowledge to critical application
- `updated_oecd_ai_lesson_plan.docx` - lesson plan with teacher-facing implementation guidance

## Topic

Implementing OECD Principles in Higher Education Classrooms, with an emphasis on responsible AI use in engineering education and faculty development.

## Notes

- The documents are intended for higher education faculty.
- They include student-friendly language, Bloom's taxonomy alignment, and practical classroom application.
- The materials also include an AI transparency statement and an ethical audit protocol.

## Suggested use

Use these documents as templates for faculty workshops, classroom planning, assessment design, and reflective practice around trustworthy AI.
